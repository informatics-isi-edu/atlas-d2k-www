--- layout: resources title: Mouse Genital Tubercule Reference Models description: A set of 3D reference models with and without the gene-expression patterns for the marker gene shh. ---

Provided by Martin J. Cohn Lab, University of Florida
March 2013 (GUDMAP)

The atlas below displays a set of 3D reference models with and without the gene-expression patterns for the marker gene Shh (sonic hedgehog).

Click on the links in the table to view the available reference models. Note the marker gene models will be spatially mapped to the wild-type embryos to enable comparison of new data against the reference pattern.
